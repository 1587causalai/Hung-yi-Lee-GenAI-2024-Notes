# 什么是生成式人工智能（GenAI）？

## 1. 人工智能的范畴

人工智能（AI）是模拟人类智能的机器。在AI的大范畴下，存在多个分支和概念：

- 人工智能 (AI)
- 机器学习 (ML)
- 深度学习 (DL)
- 生成式人工智能 (GenAI)
- 语言模型 (LM)

问题：

- 这些概念之间的具体关系和层次结构是什么？
- GenAI和传统机器学习方法有什么区别和联系？

## 2. 生成式AI的定义

根据李宏毅教授的讲解，生成式人工智能的核心能力是：
- 机器能够产生有复杂结构的物件
- 具体表现为生成文本、图像、语音、视频等内容
